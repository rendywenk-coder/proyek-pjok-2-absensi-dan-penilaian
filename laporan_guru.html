Panduan Upload Berkas

Proses Upload:

  1. Upload file ke Google Drive atau Cloud Storage lainnya
  2. Atur permission menjadi "Anyone with the link can view"
  3. Salin link file dan tempel di form di bawah
  4. Admin akan meninjau dan memberikan status DITERIMA/PERBAIKI
  5. Jika perlu perbaikan, gunakan tombol EDIT untuk memperbaiki

Tips: Gunakan format penamaan: JENIS_KELAS_SEMESTER_TAHUN.ext

Informasi Upload Soal

Format Penamaan File:

SOAL_[JENIS]_[MAPEL]_[KELAS]_[SEMESTER]_[TAHUN].pdf
Contoh: SOAL_STS_AKADEMIK_5A_S1_2024.pdf

Persyaratan:

  • Format file: PDF, DOC, DOCX
  • Maksimal ukuran: 10MB
  • Disertai kunci jawaban (jika ada)
  • Rubrik penilaian (jika diperlukan)

Perhatian: Soal yang sudah dikirim tidak bisa dihapus. Gunakan tombol EDIT jika perlu perbaikan.

Struktur Folder Rekomendasi

Untuk menjaga kerapian, gunakan struktur folder berikut di Google Drive:

├── SDN_Rawabuntu_03/
│ ├── [NAMA_ANDA]/
│ │ ├── RPP/
│ │ ├── SOAL/
│ │ ├── LEGER/
│ │ └── ADMIN/
│ └── SHARED/
│ └── (berkas bersama)
